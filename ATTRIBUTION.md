# Configuration Attribution

This file records the best available discoverer and date attribution for each configuration bundled in `optimal-solutions.generated.js`.

## Important limitations

These are historical attributions, not assertions of copyright ownership.

The primary source often records when a result was published, enumerated, or added to a symmetry class rather than when the exact configuration was first discovered. Following the working assumption used for this project, each bundled configuration is treated as the first known configuration for its grid size and encoded symmetry class.

The `rct4` class creates an additional ambiguity. Achim Flammenkamp introduced it around January 1997 after recognizing that several already-known odd-grid configurations had quarter-turn symmetry except on their long diagonals. Their original configurations can therefore predate their `rct4` classification.

Confidence labels:

- **Explicit** — the source names the discoverer and provides a date or narrow date range.
- **Inferred** — the attribution follows a dated enumeration, progress report, or named publication range.
- **Bound only** — the source establishes that the result was known by a date but does not establish its discoverer.

## Primary sources

- [Achim Flammenkamp, “The No-Three-in-Line Problem”](https://wwwhomes.uni-bielefeld.de/achim/no3in/readme.html) — historical references, format definition, and the 2025–2026 discovery chronology.
- [Progress in generating configurations with n odd](https://wwwhomes.uni-bielefeld.de/achim/no3in/odd_results.html) — creation of `rct4` and the December 1996–January 1997 odd-grid chronology.
- [New finished case studies since April 1996](https://wwwhomes.uni-bielefeld.de/achim/no3in/new_results.html) — dated even-grid searches and enumerations.
- [1992 solution-count table](https://wwwhomes.uni-bielefeld.de/achim/no3in/table_old.txt) — evidence that older size/class combinations were already known.
- [1997 solution-count table](https://wwwhomes.uni-bielefeld.de/achim/no3in/table.txt) — state of the database as of January 31, 1997.
- [Current summary table](https://wwwhomes.uni-bielefeld.de/achim/no3in/table.html) — current size/class counts and symmetry definitions.

## Corrections

Historical credit is easy to misattribute when a source reports a publication rather than a discovery. Corrections supported by a primary source are welcome. In particular, the entries marked **Bound only** and **Inferred** should not be presented elsewhere as definitive discoverer claims.
