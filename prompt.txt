Create a web app to explore the no-three-in-line problem. Use plain HTML, CSS, and JavaScript. Do not use a framework. The user can select any grid size from 3 x 3 to 68 x 68, and a grid of square cells is displayed. Initially, all cells are inactive. The user toggles the state of a cell (active or inactive) by clicking on it. The current number of active cells is displayed above the grid. If there are three or more active cells lying on a straight line, show all of the cells on that line in red.

There should be a CLEAR button, and a button to show an optimal solution for the current grid size, as well as its symmetry group. This information will be added later. If an optimal solution is not available for the current grid size then the button to show the solution is inactive.
